Apresentando Tags Básicas do HTML

O HTML (HyperText Markup Language) é a linguagem de marcação, usada para estruturar conteúdos em páginas.

Abaixo estão algumas das tags mais básicas e essenciais para começar.

Títulos (h1 a h6)

As tags de título do HTML vão de <h1> até <h6> e servem para organizar o conteúdo da página em partes com diferentes níveis de importância.

O <h1> é o título mais importante, normalmente usado para o nome da página ou o assunto principal. Depois vem o <h2>, que funciona como um subtítulo. Se você quiser dividir esse subtítulo em partes menores, pode usar o <h3>, e assim por diante até o <h6>, que é o nível mais baixo.

Resumindo: quanto menor o número, mais importante é o título. Essa hierarquia ajuda tanto os usuários a entenderem a estrutura da página quanto os mecanismos de busca (como o Google) a lerem melhor seu conteúdo.

Exemplo de como ultilizalos:

<h1>- Título

<h2>- Título

<h3>- Título

<h3>- Título

<h5>- Título
<h6>- Título

Tags <strong>, <em> e <mark>

Você pode dar ênfase ou destaque ao texto com essas tags:

  • <strong> para dar ênfase forte (semântica de importância): Importante!
  • <em> para ênfase suave (como itálico): Texto em ênfase
  • <mark> para destacar (como marca-texto): Destaque amarelo

Lista Ordenada X Lista Não Ordenada

Em HTML, as listas servem pra agrupar vários itens juntos, como se fosse uma checklist ou uma sequência. A gente geralmente usa dois tipos: a Ordenada e a lista não ordenada. Abaixo entenda as diferença entre elas:

🔢 Lista Ordenada (<ol>)

A lista ordenada usa a tag <ol> que exibe os itens em uma sequência numerada. É ideal quando a ordem dos elementos faz diferença, como em instruções passo a passo, receitas ou etapas de um processo.

Exemplo de preparo de pudim: 

                            
                                <ol>
                                 <li>Bata bem os ovos no liquidificador</li>
                                 <li>Acresente o leite condensado e o leite e bata novamente</li>
                                 <li>Derreta o açúcar na panela, acrescente água e deixe engrossar</li>
                                 <li>Coloque em uma forma redonda e despeje a massa do pudim</li>
                                 <li>Asse em forno médio por 45 minutos</li>
                                </ol>

Resultado:

  1. Bata bem os ovos no liquidificador
  2. Acresente o leite condensado e o leite e bata novamente
  3. Derreta o açúcar na panela, acrescente água e deixe engrossar
  4. Coloque em uma forma redonda e despeje a massa do pudim
  5. Asse em forno médio por 45 minutos
Voltar

● Lista Não Ordenada (<ul>)

A lista não ordenada usa a tag <ul> que mostra os itens com marcadores (normalmente bolinhas). Ela é usada quando a ordem dos itens não é importante, como em listas de países, tópicos soltos ou características.

Exemplo de lista de países: 

                            
                                <ul>
                                 <li>Brasil</li>
                                 <li>Chile</li>
                                 <li>Italia</li>
                                 <li>Inglaterra</li>
                                 <li>Japão</li>
                                </ul>

Resultado:

  • Brasil
  • Chile
  • Italia
  • Inglaterra
  • Japão

Tag <Form>

A tag <form> é usada em HTML para criar formulários, ou seja, áreas onde o usuário pode digitar informações, selecionar opções e enviar dados para um servidor.

Mas o que realmente controla o funcionamento de um formulário são os seus atributos. Abaixo conheça alguns dos principais:

name - Nome do formulário

Esse atributo dá um nome para o formulário. Ele pode ser usado para identificá-lo em scripts ou estilos, principalmente no JavaScript. 

                            
                                <form name="cadastro">
Voltar

action - Para onde os dados vão

É aqui que você define o destino do formulário, ou seja, o link (URL) para onde os dados vão ser enviados quando o usuário clicar no botão de envio. 

                            
                                <form action="/enviar-dados">
Voltar

method - Como os dados serão enviados

Define o método HTTP de envio. Os dois mais usados são:

  • GET: envia os dados na URL, usado em buscas;
  • POST: envia os dados de forma "escondida" no corpo da requisição, usado para dados sensíveis. 
                            
                                <form method="post">
Voltar

autocomplete - Completa automaticamente

Esse atributo define se o navegador pode ou não sugerir dados que o próprio usuário já preencheu antes, como nome, e-mail ou endereço. Esses dados ficam salvos localmente no navegador.

  • on: o navegador vai salvar e sugerir automaticamente
  • off: o navegador não vai sugerir nada 
                            
                                <form autocomplete="off">
Voltar

onsubmit - Ação ao enviar

Permite rodar uma funçao em JavaScript, antes dos dados serem enviados para o servidor.Essa função é útil para validar dados antes do envio, por exemplo: 

                            
                                <form onsubmit="return validarFormulario()">

Se a função validarFormulario() retornar false, o envio é cancelado.

Voltar

Exemplo do código:

                            
                                <form name="contato" action="/enviar" method="post" autocomplete="on" onsubmit="return validar()">
                                 <input type="text" name="nome" placeholder="Seu nome">
                                 <input type="email" name="email" placeholder="Seu e-mail">
                                 <button type="submit">Enviar</button>
                                </form>

Tag Table

Ao criar tabelas no HTML, é importante usar alguns elementos e atributos que ajudam na acessibilidade, facilitando a leitura por leitores de tela e a compreensão da estrutura dos dados.

<caption>

Serve como um título descritivo da tabela. Fica visível e ajuda usuários (inclusive os que usam leitores de tela) a entenderem o propósito da tabela. 

                            
                                <caption>Notas dos alunos por disciplina</caption>
Voltar

<th> + scope

<th> é usado para identificar cabeçalhos de linha ou coluna. O atributo scope define o que aquele cabeçalho representa:

  • scope="col" - Cabeçalho de coluna
  • scope="row" - Cabeçalho de linha
                            
                                <th scope="col">Aluno</th>
                                <th scope="col">Nota</th>
Voltar

headers

Usado em <td> para indicar a qual <th> ela está relacionada, especialmente útil em tabelas complexas. 

                            
                                <td headers="col1">João</td>
Voltar

<thead>, <tbody> e <tfoot>

Essas tags ajudam a separar visualmente e semanticamente o conteúdo da tabela:

  • <thead> - Parte do cabeçalho
  • <tbody> - Corpo principal da tabela
  • <tfoot> - Rodapé da tabela
                            
                                <thead>
                                 <tr>
                                  <th>Nome</th>
                                  <th>Idade</th>
                                 </tr>
                                </thead>
                                <tbody>
                                <tr>
                                  <td>Ana</td>
                                  <td>20</td>
                                 </tr>
                                </tbody>

Cores

No desenvolvimento web, as cores podem ser definidas de várias formas. Cada formato tem sua maneira de representar cor.

RGB - Vermelho, Verde e Azul

O formato rgb() define uma cor misturando os três canais de luz: vermelho (Red), verde (Green) e azul (Blue). Os valores vão de 0 a 255. 

                            
                                rgb(255, 0, 0)   → vermelho
                                rgb(0, 255, 0)   → verde
                                rgb(0, 0, 255)   → azul
Voltar

RGBA - RGB com transparência

O rgba() é como o RGB, mas com um canal extra: o alpha, que define a opacidade (de 0 a 1). 

                            
                                rgba(0, 0, 0, 0.5)   → preto com 50% de transparência
                                rgba(255, 0, 0, 0.3) → vermelho bem transparente
Voltar

HEX - Representação hexadecimal

Esse é um dos formatos mais comuns. A cor é definida por um código com 6 dígitos, representando os canais de vermelho, verde e azul. 

                            #FF0000   → vermelho
                                #00FF00   → verde
                                #0000FF   → azul

Também é possível usar um atalho com 3 dígitos: #f00 é igual a #ff0000.

Voltar

HSL – Matiz, Saturação e Luminosidade

O hsl() representa a cor de uma forma mais visual e intuitiva:

  • Hue (matiz): de 0 a 360 (ângulo da cor no círculo cromático)
  • Saturation: quão forte é a cor (0% a 100%)
  • Lightness: quão clara ou escura (0% a 100%) 
                            hsl(0, 100%, 50%)   → vermelho
                                hsl(120, 100%, 50%) → verde
                                hsl(240, 100%, 50%) → azul
Voltar

HSLA – HSL com transparência

O hsla() é igual ao HSL, mas adiciona o canal alpha (transparência): 

                            
                                hsla(0, 100%, 50%, 0.5) → vermelho com 50% de opacidade
